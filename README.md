# analise-de-faturamento


#  Projeto de Consolidação de Vendas

 # Sobre o projeto

Este projeto tem como objetivo reunir dados de vendas de diferentes meses em uma única tabela. Isso facilita a análise das informações, permitindo visualizar tudo de forma organizada e completa.

 # Objetivo principal

O principal objetivo é simplificar a análise de dados, evitando que as informações fiquem espalhadas em vários arquivos diferentes.

 Dados utilizados

Foram utilizados três arquivos de vendas, cada um representando um mês diferente:

 Vendas de Janeiro
 Vendas de Fevereiro
 Vendas de Março

Cada arquivo contém informações como produtos vendidos, valores e outras informações relacionadas às vendas.



##  O que o programa faz

O programa realiza basicamente quatro etapas:

### 1. Leitura dos arquivos

Primeiro, os arquivos de cada mês são carregados para dentro do programa.



### 2. Organização dos dados

Depois disso, os dados de cada arquivo são preparados para serem unidos.



### 3. Junção das informações

Os dados dos três meses são combinados em uma única tabela.
Isso significa que todas as vendas ficam reunidas em um só lugar, em vez de separadas por mês.



### 4. Exibição dos resultados

Por fim, a tabela final é exibida na tela para visualização.



 Resultado final

O resultado é uma tabela única contendo todas as vendas de janeiro, fevereiro e março juntas, facilitando a análise geral do desempenho das vendas.
